Learn CSS Grid

1. Grid Container

To start using CSS Grid, you need to define a grid container. You can do this by setting the display property to grid or inline-grid on an element. All direct children of this element will then become grid items.

display: grid

This creates a block-level grid container. It takes up the full width available.

1
2
3

HTML

<div class="grid-container">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>

CSS

.grid-container {
    display: grid;
    /* The following properties are for demonstration */
    /* We'll learn about them in the next sections! */
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

display: inline-grid

This creates an inline-level grid container. It only takes up as much space as its content.

Here is an inline-grid container 123 and it sits next to other text.

HTML

<p>
    Here is an inline-grid container 
    <span class="inline-grid-container">
        <span class="item">1</span>
        <span class="item">2</span>
        <span class="item">3</span>
    </span> 
    and it sits next to other text.
</p>

CSS

.inline-grid-container {
    display: inline-grid;
    /* The following properties are for demonstration */
    grid-template-columns: repeat(3, auto);
    grid-gap: 5px;
}

2. Explicit Grid

You can explicitly define the grid structure using grid-template-columns and grid-template-rows.

grid-template-columns/rows

Define the size of columns and rows. You can use any length unit like px, %, em, or the flexible fr unit which represents a fraction of the available space.

90px
1fr
2fr
90px
1fr
2fr

CSS

.explicit-grid-1 {
    display: grid;
    grid-template-columns: 90px 1fr 2fr;
    grid-template-rows: 50px 80px;
    grid-gap: 10px;
}

3. Minimum and Maximum Grid Track Sizes

The minmax() function defines a size range, greater than or equal to min and less than or equal to max.

Item 1
Item 2 has more content to demonstrate wrapping.

CSS

    display: grid;
    grid-template-columns: minmax(100px, 250px) 1fr;
    grid-gap: 10px;
}

6. Positioning Items by Grid Line Numbers

Grid lines are numbered starting from 1. You can place items precisely using grid-column-start, grid-column-end, grid-row-start, and grid-row-end, or the shorthands grid-column and grid-row.

A
B

CSS

.line-pos-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 80px);
    grid-gap: 10px;
}
.item-a {
    grid-column-start: 2;
    grid-column-end: 4;
    grid-row-start: 1;
    grid-row-end: 2;
    background-color: #8e44ad;
}
.item-b {
    /* Shorthand: row-start / col-start / row-end / col-end */
    grid-area: 2 / 1 / 3 / 3;
    background-color: #e67e22;
}

8. Naming Grid Lines

You can name grid lines to make positioning more intuitive than using numbers.

Positioned with named lines

CSS

.named-lines-grid {
    display: grid;
    grid-template-columns: [main-start] 1fr [content-start] 2fr [content-end] 1fr [main-end];
    grid-template-rows: [main-start] 100px [main-end];
    grid-gap: 10px;
}
.named-lines-item {
    grid-column-start: content-start;
    grid-column-end: content-end;
    background-color: #2980b9;
}

9. Naming & Positioning by Grid Areas

You can name entire grid areas and place items into those areas with the grid-area property. This is great for page layouts.

Header
Main Content
Sidebar
Footer

CSS

.layout-container {
    display: grid;
    grid-template-columns: 3fr 1fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "main   sidebar"
        "footer footer";
    grid-gap: 15px;
    height: 300px;
}
.item-header { grid-area: header; }
.item-main { grid-area: main; }
.item-sidebar { grid-area: sidebar; }
.item-footer { grid-area: footer; }

10. The Implicit Grid

When you place an item outside of the explicit grid, the grid creates implicit tracks to hold it. You can define the size of these tracks with grid-auto-rows and grid-auto-columns.

Explicit
Explicit
Implicit

CSS

.implicit-grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    grid-template-rows: 50px; /* Only one explicit row */
    grid-auto-rows: 100px; /* Implicit rows will be 100px tall */
    grid-gap: 10px;
}
.implicit-item {
    grid-row: 2; /* This forces it into an implicit row */
    grid-column: 1 / 3;
    background-color: #c0392b;
}

11. Layering Grid Items (z-index)

You can overlap items and control their stacking order with the z-index property, just like with regular CSS positioning.

Layer 1 (z-index: 1)
Layer 2 (default z-index)

CSS

.layering-grid {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: 100px;
    grid-gap: 10px;
}
.layer-1 {
    grid-column: 1 / 3;
    grid-row: 1;
    z-index: 1; /* Higher z-index comes on top */
    background-color: rgba(41, 128, 185, 0.8);
}
.layer-2 {
    grid-column: 2 / 4;
    grid-row: 1;
    background-color: rgba(230, 126, 34, 0.8);
}

12. Aligning Grid Items

Control the alignment of items within their grid area using justify-items (row axis) and align-items (column axis) on the container, or justify-self and align-self on individual items.

Default
align-self: center
Default
justify-self: end

CSS

.align-items-grid {
    display: grid;
    height: 200px;
    grid-template-columns: repeat(2, 1fr);
    grid-template-rows: repeat(2, 1fr);
    grid-gap: 10px;
    /* Align all items to the start of their area */
    justify-items: start; 
    align-items: start;
}
.align-self-center { align-self: center; }
.justify-self-end { justify-self: end; }

13. Aligning Grid Tracks

When the total size of the grid tracks is smaller than the grid container, you can align the tracks themselves within the container using justify-content (row axis) and align-content (column axis).

    grid-template-rows: 50px 50px;
